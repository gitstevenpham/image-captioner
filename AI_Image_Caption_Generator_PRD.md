## 10. Example API Spec

**POST** `/api/caption`  
**Request:**
```json
{
  "image_base64": "data:image/png;base64,..."
}
```
**Response:**
```json
{
  "caption": "A young woman walking through a sunflower field at sunset."
}
```

**POST** `/api/rate`  
**Request:**
```json
{
  "image_id": "abc123",
  "caption": "A young woman walking...",
  "rating": 5
}
```
**Response:**
```json
{ "status": "success" }
```
